# 🚗 Automobile Sales Analysis Dashboard

A Power BI dashboard that analyzes automobile sales, profitability, customer trends, and regional performance using sales data from 2012–2015. The project leverages Power Query for data transformation, DAX for KPI calculations, and interactive visualizations to uncover business insights that support strategic decision-making.

---

# 📌 Project Overview

This project analyzes automobile sales data using Microsoft Power BI to evaluate sales performance across brands, vehicle types, regions, customers, and time.

The dashboard transforms raw transactional data into interactive visualizations, helping identify top-performing brands, profitable markets, customer purchasing trends, and sales growth over multiple years.

# ⚙️ Data Preparation

The dataset was cleaned and transformed using Power Query.

### Data Cleaning & Transformation

- Removed duplicate records.
- Removed unnecessary columns.
- Filtered data for specific automobile brands.
- Created calculated columns such as **Total Cost**.
- Split Invoice Date into **Invoice Year** and **Invoice Month**.
- Grouped sales by Make and Country.
- Created a Budget table using DAX.
- Built KPI measures for dashboard reporting.

# 💡 Key Insights

The analysis revealed several important business insights:

- Total sales reached **32M** between 2012 and 2015.
- Sales increased steadily from **3M in 2012** to **15M in 2015**, with the largest growth occurring between 2014 and 2015.
- **Coupes** generated the highest sales, followed closely by **Saloons**, while Convertibles contributed a smaller share.
- The **DB9** was the top-performing vehicle model.
- The **United Kingdom** generated the highest sales, followed by the **United States**, together contributing over 80% of total sales.
- **Rolls Royce**, **Aston Martin**, and **Jaguar** achieved the highest profit margins.
- Actual sales consistently exceeded budget targets, particularly during the third and fourth quarters.
- Sales and profit trends followed similar upward patterns, indicating healthy profitability alongside revenue growth.
